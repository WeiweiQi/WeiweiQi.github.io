1. 调用第三方接口必须统一封装，必须获取所有可能的异常，最后的catch必须是 catch(Exception e) {...}

```Java
public Object getInfoFromOtherAPI() {
  try {
    //call API
  } catch (SpecialException e) {
    //TODO
  }
  ...
  catch (Exception e) {
    //最后的catch必须捕获所有可能的异常
    return null;
  }

}
```
